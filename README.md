## Overview

AI Resume Analyzer Agent is an offline Resume Analyzer developed using Python and Streamlit.

The application extracts text from uploaded PDF resumes and performs ATS-style analysis without using any AI API.

---

## Features

- Upload Resume PDF
- ATS Score
- Technical Skills Detection
- Soft Skills Detection
- Job Match Analysis
- Missing Skills
- Resume Strengths
- Resume Weaknesses
- Resume Recommendations
- Recommended Job Roles
- Interview Questions

---

## Technologies Used

- Python
- Streamlit
- PyPDF
- Rule-Based ATS Analysis
